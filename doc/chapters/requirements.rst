************
Requirements
************

Code-Level Requirements
=======================

* A C++ compiler compatible with C++20 or later.
* A standard C++ library supporting C++20 features.

Build Requirements
==================

* **CMake**: Version 3.23 or newer.

Test Requirements
=================

* **Python**: Version 3.12 or newer
